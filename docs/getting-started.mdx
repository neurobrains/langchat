## About LangChain

**LangChain** is a powerful framework for building applications with Large Language Models (LLMs). It provides a comprehensive set of tools, components, and integrations that make it easier to develop AI applications.

LangChat is built on top of LangChain, leveraging its robust ecosystem to provide you with a production-ready chatbot framework.

## LangChain Components in LangChat

### Document Loaders

LangChat uses LangChain's document loaders through `docsuite`:

```python
from docsuite import UnifiedDocumentLoader

# LangChain-powered document loading
loader = UnifiedDocumentLoader("document.pdf")
documents = loader.load()
```

**Supported formats:**
- PDF files
- Text files
- CSV files
- And more (via LangChain loaders)

### Text Splitters

LangChat uses LangChain's `RecursiveCharacterTextSplitter`:

```python
from langchain.text_splitter import RecursiveCharacterTextSplitter

# LangChain text splitter
text_splitter = RecursiveCharacterTextSplitter(
    chunk_size=1000,
    chunk_overlap=200
)
chunks = text_splitter.split_documents(documents)
```

### Vector Stores

LangChat integrates with LangChain's Pinecone vector store:

```python
from langchain_pinecone.vectorstores import PineconeVectorStore

# LangChain Pinecone integration
vector_store = PineconeVectorStore(
    index=pinecone_index,
    embedding=embeddings
)
```

### Embeddings

LangChat uses LangChain's OpenAI embeddings:

```python
from langchain_openai import OpenAIEmbeddings

# LangChain OpenAI embeddings
embeddings = OpenAIEmbeddings(
    model="text-embedding-3-large"
)
```

## Using LangChain Directly

While LangChat provides a high-level interface, you can also access LangChain components directly:

```python
from langchat import LangChat, LangChatConfig

config = LangChatConfig.from_env()
langchat = LangChat(config=config)

# Access LangChain components
engine = langchat.engine
vector_adapter = engine.vector_adapter

# Use LangChain vector store directly
vector_store = vector_adapter.vector_store
retriever = vector_store.as_retriever(search_kwargs={"k": 5})

# Use LangChain embeddings
embeddings = vector_adapter.embeddings
query_embedding = embeddings.embed_query("your query")
```

## Extending with LangChain

You can extend LangChat using LangChain components:

### Custom Document Loaders

```python
from langchain_community.document_loaders import PyPDFLoader

# Use custom LangChain loader
loader = PyPDFLoader("document.pdf")
documents = loader.load()
```

### Custom Text Splitters

```python
from langchain.text_splitter import CharacterTextSplitter

# Use custom LangChain splitter
splitter = CharacterTextSplitter(
    separator="\n\n",
    chunk_size=1000,
    chunk_overlap=200
)
```

### Custom Chains

```python
from langchain.chains import LLMChain
from langchain.prompts import PromptTemplate

# Create custom LangChain chain
prompt = PromptTemplate(
    input_variables=["question"],
    template="Answer: {question}"
)
chain = LLMChain(llm=llm, prompt=prompt)
```
